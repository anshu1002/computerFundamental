COMPONENTS OF COMPUTER

There are two components of computer:-

  • Hardware:-

    • Hardwarer refers to the physical components of a computer. Computer Hardware that we can touch these parts. These are the primary electronic devices used to build up the computer.
    Examples of hardware in a computer are the Processor, Memory device, Monitor, Printer, Keyboard, Mouse, and the Central Processing Unit(CPU).

  • Software:-

    • Software is a collection of instrucitons, procedures, and documentations that performs different tasks on a computer system. We can say also computer Software is a programming code executed on a computer processor.
