Input Unit

Just as we do any work or job, First of all we take instruction as per instruction be complete the job.
Similarly, Computer system also complete the task but first need to instruction or data for processing. Here instruction term accept from input unit.

Input Unit are those unit through which computer system accept data or instruction the outside world of computer system.

In a computer system, input unit accept data in various form like alphabetic/numerical i.e. textual data, sound, image or vide.

Function of Input System:-

  1. Accept data or instruction from the outside world of computer.
  2. Accept data in human readable form that convert them in machine readable/language.
  3. This machine language code pass to the CPU for further processing.

Types Of Input Devices:-

  1. Keyboard
  2. Pointing Device
  3. Scanning Device
  4. Optical Recognisation devices
  5. Media Input Device
  6. Voice Recognisation Device

KeyBoard:-

Keyboard is the most commonly use input device, we use Keyboard to type a data and text an execute command.

A Keyboard consist of following groups:-

The Alpha-Numeric Key include the number and alphabets key.

Function Key:-

THe function Key are arranged in a row on the top of keyboard. These key help perfrom specific task such as searching a file or refreshing web page, shut down key.

COntrol Key:-

Control Key also known as cursor key which control the movement of cursor on the screen. One more another name is arrow key.

Numeric Keypad:-

Numeric keypad locate on the right side of the keyboard. Its looks like a calculator keypad.

Special Purpose Key:-

It perform a some specific operation like restart, del, esc, break, etc.

Pointing Device:-

Pointing Device are generally use for moving the cursor on a particular object. That object like a meny, icon, windows etc. That are basically presenting GUI.

There are two types of Mouse:-

  1. Mechanical Mouse
  2. Optical Mouse

Mechanical Mouse consists of three buttons- lft button, Right button and scroll button. The left button use most frequently it helps in selecting the icon, folders and text on the display screen. The right button on the mouse help in display the short-cut meny which we know that pop-up menu where as the scroll button helps in moving the cursor up and doen on the screen by scrolling it.

An Optical Mouse is a ponting device. It look similar as mechanical mouse but have little bit different as per working technology. It work the reflected light determine the movement of the cursor portion of the display mouse having light emitting diode(LED) on optical sensor and a digital processor.

Scanning Device:-

Scanning device is an input device which electronically capture text & image. So convert them into capture readable form. The basic task of scanning device is convert image & textual data into digital data i..e. Represent Zero and one form.

The resultant matrix is knwon as bit map and display on the screens. The scanning devices are different each other. Some are big in size, some are can be operated by hand.

Some following characteristics:-

  1. size:- some scanning device can be easily operated by hand and some occupy more space need tp be scan big image.
  2. Resolution:-Resolution term talking about diagonal space device(length*breadth). Some device having less resolution.
  3. Technology:-There are two types of technology use in scanning device CCD(charge couple device) which is use in small scanning device and PMT(photo Multiplier Tube) use in bigger scanning device.

There are four types of scanning device:-

Handheld Scanning Device:-

The handheld scanner device are the scanning devicse that use for designing the image into bit map (which is a matrix, zeros and ones). The price and size is less used for identify the bar code level.

They are divided into two types:-

(1) Contact Scanning Device:-

Contact Scanning use by moving them like a light pen on the bar code level capture the bardcode.

(2) Non Contact scanning Device:-

Non-contact scanning device operate by pointing the gun on the barcode level captuer the image.

Flatbed Scanner:-

Flatbed Scanner having flat surface of glas which the documents are kept for sacnning. The light reflection from horizontally left to right document surfaces help in identify the image that scanning entire document line by line and this procedure repeat till the last line. The size of flatbed scanner is 8.5 inch to 11 inch (8.5"*11"). Its resolution 5500 dpi(dot per inch)

Drum Scanner:_

Drum Scanner use photo Multiplier Tube Technology. It is large drum which is used for scanning device because It is very expensive and resolution of image is very high that can scan 11"*24" and its resolution range 8000 dpi to 11000 dpi.

Slide Scanner:-

It is used for scanning slides as well as film negative are the inversion of positive image. These scanner also known as film scanner. They can easily scan original image of the film. The slide scanner are capable of scanning upto 35 slides at a time.
Its resolution range is 4000 dpi to 5000dpi.

Optical Recognisation Device:-

Optical Recognisation Device are used for recognised its individual characters and converting it into the editable form help of optically. The optical recognisation devices basically make scanner for inputing data.

The following are some optical recognisation devices are:-

Optical Character Recognisation:-

This device scan a particular document by its individual character and converting its these devics:-

  1. Image Scanner: The basic task of image scanner is to capture the text, which is to be recognised optically.
  2. OCR Software and hardware:- It process the text images captured by the image scanner but firstly it is analysed to extract the character image.
  3. Optical Interface:- It helps in establishing communication between the OCR devices and outer side world.

The Optical mark recognisation(OMR) device use of OMR technology, which helps in obtaining the data from the mark field. These devices use multiple choice question paper, enrollment form, registration form, etc.

Magnetic Ink Character Recognisation:-

MICR devices are the special devices used for recongnising the character written with magnetic in consisnting of iron oxide particles.

Bar Code Reader:-

Bar Code Reader is a kind of scanner that usually reads bar code which is use to lable scan that are repeat on the item product, wrapper basically available on super market like retail stores, books, cosmetic item, having this kind of product etc. The barcode is the computer readable code representing vertical lines for scanning the barcode which is later converted into no. or letter using decoder. First scanner scan barcode in which decoder convert bar in no. and letter and send to the computer via able for processing. Bar code reader produce light beam or bar code than reflected back from the bar code scanner sensor. Light energy is converted into electrical energy by the bar code scanner finally the electrical energy is transform into data forwarded to the computer.

Template Matching:-

First in template matching method voice is recognisation speaker dependent.
